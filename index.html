Linux Commands

cd: To change directory

ls: to list all files in the folder

mkdir: to create a new directory/folder

rm : To delete a file

rm -rf : To delete the entire folder

mv : To move file to different folder

Vi Commands

i: To get into Insert mode

esc: To get out of insert mode

Shft + colon: Command prompt for vi

w: Command to write

q: Command to quit

wq: Command to write and quit

This website is solely created for referring to useful git, vi and linux commands

Git Commands

git init: To create a repository

git add : To add the file(s) to git in preparation for a commit

git commit -m "message": To commit changes to a file

git checkout -b : To create a new branch and checkout

git checkout : To checkout branch

git merge : To merge the branch with main/master (Note: You have to be on the branch that needs to be merged)
